cfg - Folder containing configuration files
documentation - Folder containing files used to construct documentation
output - Folder containing raw output for all experiments performed during testing

A built version of the documentation is available at:
http://brianwgoldman.github.io/ReducingWastedEvaluationsCGP/
You can also check out the gh-pages branch of this repository.

For paper citation information, see:
http://liinwww.ira.uka.de/csbib?query=%22reducing%20wasted%20evaluations%20cartesian%20genetic%20programming%22

EuroGP presentation is included in this repository under the name ReducingWastedEvaluationsCGP.pptx

To run experiments, use main.py
To analyze and plot experimental data, use plotter.py
To recreate the predicted amounts of wasted evaluations plot from the paper, run wasteplot.py
