Reducing Wasted Evaluations in Cartesian Genetic Programming
===============================================================================

These pages provide code documentation for the Reducing Wasted Evaluations in Cartesian Genetic Programming paper published
in EuroGP 2013 by Brian W. Goldman and William Punch.  For information about how to run this code please see the
:mod:`cgp.main` module.  For support questions feel free to email brianwgoldman@acm.org.
